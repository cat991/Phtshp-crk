## 恋爱盲盒小程序

## 介绍
微信小程序，使用vant组件库，小程序云开发。
随机抽取一张纸条，开始你们的邂逅吧



## 功能
抽取纸条
放入纸条
查看纸条
查看消费
~~获取金币~~
~~纸条分类~~
